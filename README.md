# WebCal Viewer

WebCal / iCal 形式のカレンダーを表示するウェブページです。ファイルはクエリで複数指定でき、URL を登録しておくことで自分の予定を一覧する Web ページを作成できます。

iCloud カレンダーからの公開 URL で動作を確認しています。
